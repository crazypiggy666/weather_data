# weather_data
Scripts of downloading weather data (satellite and reanalysis data)

## ERA5

Supported data:

- Pressure level data
- Model level data

## H8: Himawari-8 Level 2 data

Supported products:

- Aerosol Property (ARP)
- Cloud Property (CLP)
- Photosynthetically Available Radiation (PAR)


## FY

Supported data:

- All ordered data on the FY satellite website
